Box display types

CSS has other types of boxes that behave differently with block level element box. The type of box applied to an element is specified by the display property.

Common display types

The most common values available for display are block, inline, and inline-block.

  • block

    • Defined as a box that's stacked upon other boxes, and can have width and height set on it.The whole box model applies to block boxes.

  • inline

    • It is the opposite of a block box.Width and height settings have no effect on inline boxes, any padding,margin and border set on inline boxes will update the position of surrounding text, but will not effect the position of surrounding block boxes.

  • inline-block

    • It is something in between the first two

Uncommon display types

  • display: table
  • display: flex
  • display: grid
