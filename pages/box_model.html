Beyond setting the width, height, border, padding and margin of boxes, there are some other properties available to change how they behave.

Overflow

When setting the size of a box with absolute values, the content may overflow the box. We can use the overflow property to control this happen.It takes several possible values, most common are:

  • auto: overflowing content is hidden, scroll bars are shown.
  • hidden: overflowing content is hidden
  • visible: overflowing content is shown outside of the box

Background clip

backgrounds(background-color, background-image)extend to the outer edge of the border. This behaviour can be adjusted by setting the background-clip property on the box

Background clip

Outline

the outline of a box is not the part of box model.It is drawn outside the border box, inside the margin area.

Types of CSS boxes

Boxes represent block level elements.The type of box applied to an element is specified by the display property. The most common values of display property are:

  • block: content before and after the box appears on a separate line
  • inline: it is the opposite of a block box.Width and height settings have no effect on inline boxes. Content will appear on the same line as surrounding text and other inline elements
  • inline-block: between the first two. If content is short it is same to inline box; if content is long it is same to block box
