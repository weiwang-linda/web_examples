background and it's property

The background of an element is the area that sits underneath an element's content, padding,and border.It can be alter with the background-clip property.

  • background-color
  • background-image
  • background-position
  • background-repeat
  • background-attachment
  • background
  • background-size

background color

The default background color of most elements is not white but transparent

background image

The simplest usage of this property involves using the url(pathofimage) function

background repeat

  1. no-repeat
  2. repeat-x
  3. repeat-y

background position

  • Absolute values like pixels
  • Relative values like rems
  • Percentages
  • keywords: right, center, top, left, bottom

background image:gradients

There are two types: linear gradients and radial gradients

linear gradients pass a linear-gradient()function to background-image property.The function needs to take three parameters separated by commas.

  1. the direction the gradient should be going in across the background
  2. the color at the beginning
  3. the color at the end

We can specify the color stops along the gradient
