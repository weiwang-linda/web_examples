The cascade

The cascade indicates that the order of CSS rules matter, but it's more complex than that.There are three factors the cascade depends on.

  1. Importance
  2. Specificity
  3. Source order

Importance

There is a special piece of syntax which to make sure that a certain rule will always win over all others: !important

One selector to rule them all!

Specificity

From example we can see the priority of some specificity: element selectors < class selectors < ID selectors < !important

The amount of specificity a selector has is measured using four different values, four single digits in four columns(thousands, hundreds, tens and ones)

  1. Thousands: Score one in this column if the declaration is inside a style attribute, otherwise 0.
  2. Hundreds: Score one in this column for each ID selector contained inside the overall selector.
  3. Tens: Score one in this column for each class selector, attribute selector, or pseudo-class contained inside the overall selector.
  4. Ones: Score one in this colum for each element selector or preudo-element contained inside the overall selector
  5. Universal selector(*), combinators(+,>,~,'') and negation pseudo-class(:not) have no effect on specificity

Source order

Source order means later rules will win over earlier rules.

Inheritance

  • It makes sense for font-family and color to be inherited
  • It makes sense for margin, padding, border, and background-image to NOT be inherited.

Controlling inheritance

  • inherit: This value sets the property value applied to a selected element to be the same as that of its parent element.
  • initial: This value set the property value applied to a selected element to be the same as the value set for that element in the browser's default style sheet.If no value is set by the browser's default style sheet and the property is naturally inherited, then the property value is set to inherit instead.
  • unset: This value resets the property to its natural value, which means that if the property is naturally inherited it acts like inherit, otherwise it acts like initial.
