Head's content is not displayed on the page.Head's job is to contain metadata about the document.

What is metadata?

Metadata is data that describes data. e.g: who write the document, and its summary

  • Adding title

    • <title> element can be used to add a title to the webpage. It is not the doucument's content. It is different with <h1>. title content will display as bookmark name

  • The <meta> element

    • The offical way of adding metadata to a document is the <meta> element. There are a lot of different types that can be included in page's <head>.

    1. Document's character encoding

      2. <meta charset="utf-8"> specifies the document's character encoding is utf-8.

    2. Adding an author and description

      3. Using name and content attributes in <meta>

      name: the type of meta element it is, what type of information it contains ; content: the actual meta content.

    3. Other types of metadata

      4. Using property attribute for metadata to display for facebook and twitter

      <meta property="og:description" content="some text"> ; <meta property="twitter:title" content="title text">

  • Adding custom icons to your site

    • A 16 X 16 pixel icon is used in multiple places. It will display in the browser tab containing each open page, and next to bookmarked pages in the bookmarks panel. Saving it.ico in the same directory as site's index page.

    <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

  • Applying CSS and JavaScript to HTML

    • CSS use <link> element; rel="stylesheet"---indicates that it is the document's stylesheet. e.g <link rel="stylesheet" href="xxx.css">

    JavaScript use <script> element does not have to go in the head, it is better to put it at the bottom of the document body part. <script src="xxx.js"></script>

  • Setting the primary language

    • Adding the lang attribute to html tag. e.g: <html lang="en-US"; it also can be used in subsections of the docuement for different languages
