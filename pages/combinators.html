Combinators rules

combinators allow you to combine multiple selectors together, there are four types:

  • The descendant selector (space) : allows you to select an element nested somewhere inside another element, include direct descendant or grandchild.
  • The child selector > : allows you to select an element that is an immediate child of another element.
  • The adjacent sibling selector + : allows you to select an element that is an immediatte sibling of another element, right next to it, at the same level in the hierarchy
  • The general sibling selector ~ : allow you to select any elements that are siblings of another element, at the same level in the hierarchy, but not necessarily right next to it.
